Windows下搭建nodejs与Electron开发环境

2021年8月27日 10:49:19

安装Nodejs

    从https://nodejs.org/dist/v14.17.6/node-v14.17.6-x64.msi下载nodejs安装程序。

配置下载源

    因为可能的GFW问题（不然会下载很慢很慢，也可能下载失败），所以建议把npm的仓库切换到国内taobao仓库，执行下面的命令：

npm config set registry "https://registry.npm.taobao.org/"

还有一种方式，注册cnpm命令，如下（我采用的是第二种）：

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装Electron

npm install -g electron
//默认cnpm是没有安装的，见后安装方法。
//cnpm install -g electron

查看electron是否安装成功可通过命令 electron -v 查看。

错误处理

    当出现错误：

……Electron RequestError: read ECONNRESET……

    可以考虑使用CNPM安装解决。

cnpm安装

// 指定淘宝仓库地址
npm install -g cnpm --registry=https://registry.npm.taobao.org

// 查看cnpm安装信息
cnpm -v

// 安装
cnpm i -D electron electron-reload

打包输出工具

    为了方便最终成果输出，建议安装electron-packager工具，安装也很简单，建议以下面的命令全局安装：

npm install -g electron-packager

其它工具

    安装其它如vscode、git工具等，至此实际上开发环境已经搭建好了。
