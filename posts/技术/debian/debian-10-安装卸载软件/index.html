debian 10 安装、卸载软件

2021年9月14日 10:49:19

    apt-get是Debian系统下进行软件管理的工具，可以轻松安装一些软件，可以自动卸载软件。

安装软件


apt-get install 软件名

#或者

dpkg -i  软件名

更新已安装包

apt-get upgrade

卸载软件


#不删除软件的配置文件
apt-get remove 软件包

#删除相应的配置文件
apt-get remove --purge 软件包

例：

#卸载mysql
apt-get remove mysql-server
#但是mysql相关的配置文件，将不会被删，如果再想安装，可能会出问题。比如配置文件已存在，现在无法安装等问题。

#执行
apt-get autoremove
#将依赖的软件包卸载掉，这样就可以完全卸载一个软件

#通过“apt-get remove mysql-server”命令和“apt-get autoremove”命令可以将mysql软件彻底删除

#运行
dpkg --list | grep mysql
#查看mysql有哪些安装还没有完全卸载
