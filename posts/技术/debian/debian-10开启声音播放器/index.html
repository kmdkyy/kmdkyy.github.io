开启声音，播放器

2021年8月27日 10:49:19

开启声音

安装alsa-utils

apt-get install alsa-utils

设置音量，保存

alsactl store

设置音量

amixer set Master 85%

初始化

alsactl init

安装播放器

apt-get install sox

安装所有支持的格式

apt-get install sox libsox-fmt-all

播放系统中所有音频文件，我们可以采用这个命令：

play *mp3

如果要播放某个特定的文件，那么采用这个命令：

play alarm.wav
