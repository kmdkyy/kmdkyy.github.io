linux 谷歌浏览器参数大全

2021年8月27日 10:49:19

安装谷歌浏览器

apt-get install fonts-liberation
dpkg -i google-chrome-stable_current_amd64.deb

在root用户下不支持沙盒，可以加–no-sandbox启动

google-chrome --no-sandbox

在root下打开chrome会出现no sandbox的错误的解决方案

  1. 找到google-chrome文件，在目录/opt/google/chrome 下

  2. 使用gedit打开该文件

    最后一行有

    exec -a "0""
    HERE/chrome" "$@"

    将其修改为

    exec -a "0""
    HERE/chrome" "$@" --user-data-dir --no-sandbox
  1. 浏览器启动参数
exec google-chrome \
    --kiosk --start-fullscreen --incognito \
    --no-first-run \
    --disable-session-crashed-bubble \
    --disable-background-networking \
    --disable-infobars \
    --disable-restore-background-contents  \
    --disable-translate  \
    --disable-new-tab-first-run \
    --noerrdialogs \
    --no-sandbox \
    --incognito \
    --user-data-dir=/tmp \
    --window-size=1600,900 \
    http://localhost:3000

  1. 因为echarts在多点触摸屏中，柱状图放大后不能左右拨动，还需要在启动时添加–touch-events启动参数

  2. 注意添加–incognito参数，防止在重启后google-chrome浏览器提示恢复页面。
